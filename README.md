# Simple-PHP-Login-API  
It's a easy login api.  
It's very easy to use.  

## Requirements
- **PHP >= 7.0**
- **MySQL >= 5.5**
  
## Quick Start  
1. Configure your database information in `config.php`.  
2. Navigate to `http://example.com/install.php` in your browser.  
3. Start using it!
